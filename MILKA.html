Introducción

Aquí empezamos en las clases presenciales a realizar dinámicas para poder realizar propuestas de solución innovadoras que afronten ODS y de manera virtual empezamos a aprender a cómo desarrollar nuestra bitácora Digital.

Clase Presencial

En la clase, vimos una presentacion sobre ¿Qué es una bitácora digital? y ¿Por qué tener una?. Esta presentación la tuvimos que ver en las clases virtuales pero por feriado no se pudo.

Después de la presentacion sobre las Bitácoras Digitales, entramos al Diseño Web.
Las capas que tiene el diseño Web (HTML, CSS, JAVASCRIPT) No hemos ahondado mucho en CSS ni JAVA, por temas de complejidad y tiempo. Pero si en HTML, vimos la estructura de un elemento en HTML y que significa cada una de sus partes.
La capa HTML la utilizaremos en el Visual Studio Code para nuestra bitácora digital y veremos más adelante un poco de CSS para diseño.

Actividades

Finalmente, el profesor nos presentó que a parte de la bitácora digital tendremos unos proyectos que entregar semanalmente como grupo en diferentes fechas.
