Semana 03

Aquí te voy a contar todo de la semana 03 sobre
Soluciones innovadoras a problemáticas ODS y programación en HTML.

Introducción

En esta semana logramos ver propuestas que ya existen en nuestro país que son proyectos tecnológicos de gran impacto que resuelven problemáticas de la ODS y también logramos empeazar a aprender programación HTML.

Clase Virtual

En esta clase empezamos descargando un programa llamado Git Setup Wizard, programa que integraremos con Visual Studio Code.
Luego abrimos Visual Studio Code y nos dirigimos a la parte de arriba con la lupita, ahí colocaremos nuestor dominio sacado de Github el repositorio ya creado.
Esto va a hacer que nuestros archivos subidos en github se clonen en Visual Studio, la diferencia es que será en código, ya que ahí haremos los cambios.

Terminando esto, veremos como funciona Visual Studio Code que ahora está vinculado con nuestra cuenta de Github. Primero abrimos el index desde Run Without Debugging, ahí veremos los cambios que vayamos haciendo en el código de Visual.
Iremos probando con pequeños cambios para ver como funciona, también como SUBIR los cambios a nuestro github. Es ahí donde los cambios que hagamos de manera local, se subirían a la plataforma web para todo público.

Clase Presencial

En esta clase, nos dieron mayor información sobre HTML, como es su interpretación en CSS y también hicimos un repaso de lo visto en la clase virtual sobre VS Code apoyados en la plantilla.
El profesor nos mostró denuevo paso a paso como clonar archivos a Github y que aparezcan en Visual Studio.

Vimos como utilizar Chat GPT como herramienta de apoyo para códigos en HTML. Por último, el profesor nos pidió que hicieramos cambios en la bitácora digital que estaba proyectando para que así apliquemos lo aprendido.
